 # Welcome to AcademicHub 🎓

AcademicHub is a cutting-edge university school web application designed to streamline academic management and enhance student engagement. Built with modern technologies including Vite, React, Node.js, Chart.js, MySQL database, Express.js, Nodemon, Tailwind CSS, and Heroicons, AcademicHub aims to revolutionize how educational institutions manage their operations.

**Note**: This is a client project, and due to confidentiality agreements, I am unable to provide access to all necessary files and the database.

## Features

- **Interactive Dashboard**: Visualize key academic metrics and student performance trends using dynamic charts powered by Chart.js.
- **Student Management**: Efficiently manage student profiles, attendance, grades, and academic progress.
- **Course Administration**: Streamline course scheduling, curriculum management, and assignment tracking.
- **Collaborative Tools**: Facilitate seamless communication between faculty, students, and administrators through integrated chat features.
- **User Modules**: AcademicHub supports three types of users:
  - **Public User**: Accesses general information about the institution, such as courses offered and events.
  - **Admin User**: Manages administrative tasks, including user management, course scheduling, and system configurations.
  - **Customer Representative**: Employees of the organization who assist users with queries, manage support tickets, and provide personalized assistance.

## Development Status

AcademicHub is currently in active development:
- **Frontend**: The frontend development is nearing completion, focusing on creating a visually appealing and user-friendly interface.
- **Backend**: Backend development is underway, leveraging Node.js and Express.js for robust server-side operations. Nodemon ensures smooth development workflows with automatic server restarts.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the frontend development server with `npm run dev`.
4. Start the backend server with `npm start`.
5. Access AcademicHub at `http://localhost:5173` in your browser.
